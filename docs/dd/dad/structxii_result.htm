Detailed Description

Default enum for returning failure or success, instead of using a bool.

void xiiResult::AssertSuccess ( const char *  msg = nullptr,
const char *  details = nullptr 
) const

Asserts that the function succeeded. In case of failure, the program will terminate.

If msg is given, this will be the assert message. If details is provided, msg should contain a formatting element ({}), e.g. "Error: {}".

◆ operator bool()

xiiResult::operator bool ( ) const
inlineexplicit

Same as 'Succeeded()'.

Allows xiiResult to be used in if statements:

  • if (r)
  • if (!r)
  • if (r1 && r2)
  • if (r1 || r2)

Disallows anything else implicitly, e.g. all these won't compile:

  • if (r == true)
  • bool b = r;
  • void* p = r;
  • return r; // With bool return type
