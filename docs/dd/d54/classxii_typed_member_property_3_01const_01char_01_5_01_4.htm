Detailed Description

Specialization of xiiTypedMemberProperty for const char*.

This works because xiiTypedMemberProperty< typename xiiTypeTraits<Type>::NonConstReferenceType > in xiiAccessorProperty does not actually remove the constness of the type but of the pointer, so const char* is not affected.
