◆ AddCommonStreams()

void xiiMeshBufferResourceDescriptor::AddCommonStreams ( )

Adds common vertex streams to the mesh buffer.

The streams are added in this order (with the corresponding stream indices):

  • Position (index 0)
  • TexCoord0 (index 1)
  • Normal (index 2)
  • Tangent (index 3)
