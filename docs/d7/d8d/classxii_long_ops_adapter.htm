#include <LongOpsAdapter.h>

Detailed Description

This singleton lives in the editor process and monitors all xiiSceneDocument's for components with the xiiLongOpAttribute.

All such components will be automatically registered in the xiiLongOpControllerManager, such that their functionality is exposed to the user.

Since this class adapts the components with the xiiLongOpAttribute to the xiiLongOpControllerManager, it does not have any public functionality.
