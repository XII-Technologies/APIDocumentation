xiiUInt64 xiiGALBufferCreationDescription::m_uiImmediateContextMask = XII_BIT(0)

Defines which immediate contexts are allowed to execute commands that use this buffer. The default is the main immediate context. Only specify the bits that indicate those immediate contexts where the resource will be used, setting unnecessary bits will result in extra overhead.
