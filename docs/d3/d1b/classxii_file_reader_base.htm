Detailed Description

The base class for all file readers. Provides access to xiiFileSystem::GetFileReader, which is necessary to get access to the streams that xiiDataDirectoryType's provide. Derive from this class if you want to implement different policies on how to read files. E.g. the default reader (xiiFileReader) implements a buffered read policy (using an internal cache).
