Detailed Description

Base class for settings components, of which only one per type should exist in each world.

Settings components are used to store global scene specific settings, e.g. for physics it would be the scene gravity, for rendering it might be the time of day, fog settings, etc.

Components of this type should be managed by a xiiSettingsComponentManager, which makes it easy to query for the one instance in the world.
