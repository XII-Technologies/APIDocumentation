Detailed Description

template<typename DataType, bool ReturnsIndex = true, typename ReturnType = DataType, typename StorageType = typename xiiBitIteratorStorage<DataType>::Type>
struct xiiBitIterator< DataType, ReturnsIndex, ReturnType, StorageType >

Configurable bit iterator. Allows for iterating over the bits in an integer, returning either the bit index or value.

Template Parameters
DataTypeThe type of data that is being iterated over.
ReturnsIndexIf set, returns the index of the bit. Otherwise returns the value of the bit, i.e. XII_BIT(value).
ReturnTypeReturned value type of the iterator. Defaults to same as DataType.
StorageTypeThe storage type that the bit operations are performed on (either xiiUInt32 or xiiUInt64). Auto-computed.
