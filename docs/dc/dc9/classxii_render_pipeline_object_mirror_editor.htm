Detailed Description

This custom mirror additionally sends over the connection meta data as properties to the engine side so that the graph can be reconstructed there. This is necessary as DocumentNodeManager_DefaultConnection does not contain any data, instead all data is in the DocumentNodeManager_ConnectionMetaData object.
