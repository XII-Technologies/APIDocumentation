enum xiiGALCullMode::Enum

Defines which sides of a polygon gets culled by the graphics card.

Enumerator
None 

Triangles do not get culled.

Front 

When the 'front' of a triangle is visible, it gets culled. The rasterizer state defines which side is the 'front'. See xiiGALRasterizerStateCreationDescription for details.

Back 

When the 'back' of a triangle is visible, it gets culled. The rasterizer state defines which side is the 'front'. See xiiGALRasterizerStateCreationDescription for details.
