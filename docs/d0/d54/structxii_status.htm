xiiStatus::operator bool ( ) const
inlineexplicit

Same as 'Succeeded()'.

Allows xiiStatus to be used in if statements:

  • if (r)
  • if (!r)
  • if (r1 && r2)
  • if (r1 || r2)

Disallows anything else implicitly, e.g. all these won't compile:

  • if (r == true)
  • bool b = r;
  • void* p = r;
  • return r; // with bool return type
