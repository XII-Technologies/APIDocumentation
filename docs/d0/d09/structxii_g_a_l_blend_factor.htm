DestColor 

The blend factor is RGB data from a render target.

InvDestColor 

The blend factor is One minus RGB, where RGB is the data from a render target.

SrcAlphaSaturated 

The blend factor is (f, f, f, 1), where f = min(As, 1 - Ad). As is alpha data from a pixel shader, and Ad is alpha from a render target.

BlendFactor 

The blend factor is the constant blend factor set with the xiiGALRenderCommandEncoder::SetBlendState(...).

InvBlendFactor 

The blend factor is One minus the constant blend factor set with the xiiGALRenderCommandEncoder::SetBlendState(...).

SrcOneColor 

The blend factor is the second RGB data output from a pixel shader.

InvSrcOneColor

SrcOneAlpha
