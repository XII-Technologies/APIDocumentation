Constructor for the xiiIReflectedTypeAccessor.

It is a valid implementation to pass an invalid handle. Note that in this case there is no way to determine what is actually stored inside. However, it can be useful to use e.g. the xiiReflectedTypeDirectAccessor to set properties on the engine runtime side without having the xiiPhantomRttiManager initialized.
