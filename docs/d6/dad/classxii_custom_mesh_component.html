Detailed Description

This component is used to render custom geometry.

Sometimes game code needs to build geometry on the fly to visualize dynamic things. The xiiDynamicMeshBufferResource is an easy to use resource to build geometry and change it frequently. This component takes such a resource and takes care of rendering it. The same resource can be set on multiple components to instantiate it in different locations.
