Detailed Description

A utility to gather raw geometry from a world.

The utility sends xiiMsgExtractGeometry to world components and they may fill out the geometry information. ExtractionMode defines what the geometry is needed for. This ranges from finding geometry that is used to generate the navmesh from to exporting the geometry to a file for use in another program, e.g. a modeling software.
